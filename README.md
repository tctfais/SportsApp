# SportsApp
<H1>Описание</H1>
Приложение для большой транснациональной компании, производящей спортивные товары (одежда, обувь) и инвентарь. Для популяризации и продвижения своей продукции менеджмент решает, что необходимо освоить новые каналы для информирования покупателей о выходе новых товаров и стимулировать спрос на новые вещи. В компании уже разработаны приложения для покупки товаров, а также узкоспециализированные приложения для некоторых видов спорта. 

<H1>Основные требования</H1>
<ol>
    <li>Реализация социальных компонентов в приложении. Оно должно формировать социальные группы по интересам, которые будут самоподдерживаться и в которых участники будут общаться и влиять друг на друга. Одна из целей — формирование такого образа бренда в глазах участников, который позволит нашим товарам оставаться в фаворитах при выборе из прочих равных. 
    <li>Предоставление информации о характеристиках тренировки, сравнение с прошлыми тренировками, сравнение с людьми в регионе, с профессиональными спортсменами. Основное сравнение — с самим собой для стимулирования результатов</li>
    <li>Возможность поиска людей по схожим интересам, поиск людей, которые тренируются в том же месте, имеют те же маршруты или которые тренируются прямо сейчас, для формирования групп для совместных занятий</li>
    <li>Возможность указания своего спортивного инвентаря (обувь, снаряды) для подсказок по составлению тренировок или своевременного обновления обуви</li>
    <li>Формирование и подсказки по составлению тренировок и их расписания</li>
    <li>Уведомление друзей о ваших новых успехах</li>
    <li>Геймификация</li>
    <li>Внедрение промоакций и новостей спорта в зависимости от характера тренировок для вовлечённости</li>
    <li>Возможность подключения сторонних устройств для отслеживания тренировок (датчик сердцебиения, кислорода и так далее)</li>
    <li>Лёгкая интеграция существующих приложений компании для облегчения продаж</li>
    <li>Возможность вставки региональных промоакций</li>
    <li>Возможность проведение соревнований с большим количеством участвующих</li>
    <li>Подключение дополнительных устройств, которые помогают следить за состоянием организма, для продвинутых спортсменов</li>
    <li>Интеграция с фитнес-функциями телефона</li>
</ol>

<H1>Бизнес цели</H1>
Компания стремится предоставить каждому спортсмену — от профессиональных спортсменов и любителей бега и йоги до детей на детской площадке — возможность, продукцию и вдохновение для достижения спортивных целей, на которые каждый способен. У любого человека есть потенциал для великих свершений. Если у Вас есть тело - значит Вы спорстмен. Исходя из этого можно сформировать основые бизнес-цели:
<ol>
    <li>Популяризация и стимулирование регулярного занятим спортом</li>
    <li>Создание спортивной экосистемы</li>
    <li>Повышения удобства подбора спортивного инвентаря</li>
    <li>Освоение новых каналов и способов информирования о новых товарах</li>
</ol>

<H1>Функциональные требования</H1>
<ol>
    <li>Приложение должно иметь сквозную авторизацию с другими приложениями компании</li>
    <li>Приложение должно иметь возможность получать данные доступные из финтес-устройств пользователя</li>
    <li>Приложение должно иметь возможность получать данные доступные из финтес-функциям телефона пользователя</li>
    <li>Данные от фитнес-устроств должны иметь возможность быть проанализированые</li>
    <li>Приложение должно давать возможность пользователя узнавать о проводимых соревнованиях</li>
    <li>Административное приложение должно иметь возможность создавать мероприятия (соревнования)</li>
    <li>Приложение должно уметь сообщать пользователя о проводимых промоакциях с учетом профиля пользователя</li>
    <li>Приложение должно иметь возможность бесшовно переходить к приложениям для продаж товаров</li>
    <li>Приложение должно уметь анализировать и присваивать спортивный профиль пользователя на основе его тренировок и предпочтений</li>
    <li>Приложение должно уметь отображать промоакции и новости спорта в зависимости характера тренировок пользователя</li>
    <li>Приложение должно иметь функции геймификации</li>
    <li>Приложение должно иметь возможность добавлять в друзья других пользователей</li>
    <li>Приложение должно иметь возможность приглашать других людей в приложение</li>
    <li>Приложение должно иметь возможность отправлять и получать уведомления о достижениях друзей</li>
    <li>Приложение должно иметь возможность интегрироваться с календарем пользователя</li>
    <li>Приложение должно иметь возможность помогать пользователю составлять расписание тренировок и их состав</li>
    <li>Приложение должно иметь возможность запоминать спортивный инвентарь пользователя для подсказок по составлению тренировок или своевременного обновления спортивного инвентаря</li>
    <li>Приложение должно иметь возможность поиска других пользователей по месту, спортивным интересам или схожим маршрутам тренировок</li>
    <li>Приложение должно иметь возможность создавать совместные тренировки и приглашать на них других пользователей</li>
    <li>Приложение должно иметь возможность скрытия/отображения профиля пользователя для поиска другими людьми</li>
    <li>Приложение должно иметь возможность запоминать и отображать тренировки пользователя</li>
    <li>Приложение должно иметь возможность пользователю сравнивать свои тренировки с предыдущими</li>
    <li>Приложение должно иметь возможность пользователю сравнивать свои тренировки с тренировками других пользователей</li>
    <li>Приложение должно иметь возможность пользователю сравнивать свои тренировки с тренировками профессиональных спорстменов</li>
    <li>Приложение должно давать пользователю возможность создавать и модерировать сообщества по интересам</li>
    <li>Приложение должно давать советовать товары пользователям в сообществах, соревнованиях, меропрятиях</li>
</ol>
